---
description: '为当前选中的代码行或脚本文件添加注释'
---

0. 作用域目标文件：.ts .js .vue .html .css .scss .sass .less , 其他文件 暂不支持
1. 为当前选中的代码行或脚本文件添加注释
2. 注释内容将添加到代码行或脚本文件中
3. 等待用户确认
4. 用户确认后，注释将被添加到代码行或脚本文件中
5. 用户取消后，注释将不会被添加到代码行或脚本文件中

注意：

0. 生成的注释直接更新到代码行或脚本文件中，等待用户确认合并
1. 如果当前选中的代码行或脚本文件中已经有注释，则会更新并覆盖原有注释
2. 如果选择的是文件，则会为文件添加注释
3. 如果选择的是文件夹，则会为文件夹中的所有文件添加注释
4. 如果选择的是代码块/函数/变量/类，则只为选中的代码块添加注释
5. 函数注释的格式为：

```
/**
 * 函数名称
 * 函数描述
 * @param {参数类型} 参数名称 参数描述
 * @return {返回值类型} 返回值描述
 * @example 示例代码
 * @throws {异常类型} 异常描述
 */
```

> 适用于 .ts 和 .js 文件 以及 .vue 文件中 script 标签中的代码块

---

类注释的格式为：

```
/**
 * 类名称
 * 类描述
 * @example 示例代码
 */
```

> 适用于 .ts 和 .js 文件 以及 .vue 文件中 script 标签中的代码块

---

js / TS / css (scss/sass/less) 文件注释的格式为：

```
/**
 * 文件名称
 * 文件描述
 * @author 作者， 默认为 Sogrey
 * @date 创建日期， 默认为 2025-06-01 00:00:00 （只在第一次创建时添加更新）
 * @lastModify 最后修改日期， 默认为 2025-06-01 00:00:00 （每次更新时添加更新）
 * @version 版本号
 */
```

> 适用于 .ts 和 .js 文件 以及 .vue 文件中 script 标签中的代码块

---

html / vue 文件注释的格式为：

```
<!--
* 文件名称
* 功能描述
* @author Sogrey
* @date 2025-11-08
* @lastModify 2025-11-08
* @version 1.0.0
-->
```

> 适用于 .html 和 .vue 文件
